Para correr el backend ejecutar de la terminal en la carpeta "server" el comando "npm start".
Para testear la API ejecutar "npm test" en la misma "server" ya corriendo el server.
Para correr el frontend ejecutar "npm start" en la carpeta client.

Tambien se puede correr el back y el front con docker ejecutando "docker-compose up --build" en la carpeta "code-challenge-toolbox-juan-carrillo"
